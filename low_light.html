Enhance Low-Light Images

  • Histogram Equalization: Utilize the "Night_Enhancement_by_Histogram_equalization.py" script to enhance low-light images by adjusting their histograms.
  • KS Algorithm: Employ the custom-developed "Night_KS_algorithm.py" to enhance very low-light images using a specialized algorithm
  • Parameter Tuning: Fine-tune parameters for different types of low-light images, such as medium and very low-light conditions.
  • Deep Learning Model Training: Train a deep learning model on a custom low-light image dataset. Start by running the "model.py" file to create the model and then execute the "deblur_ae.py" file for deblurring.
