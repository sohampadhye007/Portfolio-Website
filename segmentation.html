Road Segmentation

The "Road Segmentation for Autonomous Vehicles" project tackles the challenge of segmenting various road types, including off-road, Indian roads, and low-light conditions, to facilitate autonomous vehicle development. This project employs semantic and instance segmentation techniques.

Semantic Segmentation with KNN and GMM:

Initially, the project utilizes K-Means (KNN) and Gaussian Mixture Models (GMM) for basic semantic segmentation, clustering pixels based on color and texture. While effective for simple road images, these methods struggle with complex scenarios.

Semantic Segmentation with UNet:

The UNet architecture is employed for precise semantic segmentation, classifying each pixel into predefined categories. The model is trained on the custom "IITJ_RoadSeg" dataset.

Instance Segmentation with YOLOv8:

Instance segmentation, detecting and segmenting objects, is accomplished using YOLOv8 on a custom dataset from IIT Jodhpur. This is crucial for recognizing roads, vital for autonomous vehicles.

Performance and Results:

  • YOLOv8 Instance Segmentation: Mean Average Precision (mAP50): 0.908
  • UNet Semantic Segmentation: Average Loss: 0.03
